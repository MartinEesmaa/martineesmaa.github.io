VISQOL testing for points

Before I start to test, I need a lossless audio file, AC3 programs (aften and liba52) and VISQOL

Used Sintel (2009, Blender movie) trailer, Aften 0.0.8 (encoder) and liba52 0.7.4 (decoder).

Download:

Audio file [FLAC] Aften liba52 brought with a52dec VISQOL Visqol Analyze batch script (Windows) VISQOL JSON reports

Commands used:

aften -b 128 -w 60 sintel_trailer-audio.wav sintel_trailer-audio.ac3

a52dec -o wav sintel_trailer-audio.ac3 > degraded.wav

visqol --reference_file sintel_trailer-audio.wav --degraded_file degraded.wav

ffmpeg -i sintel_trailer-audio.flac -c:a ac3 -b:a 128k -cutoff 24000 sintel_trailer-audio.ac3
